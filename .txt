# 📝 App Description for Developer Reference

*App Name Rise30:* FocusFlow – 30-Day Challenge Builder

*Core Concept:*
A mobile-first app that helps users build habits, improve skills, or boost productivity via structured daily challenges. The app emphasizes *streaks, micro-dopamine rewards, and adaptive difficulty* to maximize retention. Users can start with 7, 14, or 30-day challenges and optionally invite friends for accountability.

*Primary Goal:*
✅ Users complete at least 7 consecutive days of their chosen challenge. Retention metrics are the single most important success metric.

# ⚙️ Technical Stack

### Frontend

* *Mobile Client:* Kotlin (for Android; optionally use Kotlin Multiplatform for iOS later)
* *UI Framework:* Jetpack Compose (modern, reactive UI)
* *Notifications:* Firebase Cloud Messaging (FCM) for push notifications
* *Local Storage:* Room DB (for offline caching of streaks and daily tasks)

### Backend

* *API Layer:* Express.js (Node.js)
* *Database:* Supabase (PostgreSQL) with Prisma ORM
* *Authentication:* Supabase Auth (email/password, social logins optional)
* *Hosting:* Vercel / Railway (for Express backend)
* *Analytics:* Mixpanel / Supabase built-in logs

### Data Modeling (Prisma Schema Concept)

prisma
model User {
  id            String      @id @default(uuid())
  email         String      @unique
  password      String
  createdAt     DateTime    @default(now())
  challenges    Challenge[]
  streaks       Streak[]
  dailyEntries  DailyEntry[]
}

model Challenge {
  id          String      @id @default(uuid())
  userId      String
  name        String
  type        String      // skill, productivity, fitness
  duration    Int         // days (7/14/30)
  startDate   DateTime
  endDate     DateTime
  dailyTasks  DailyEntry[]
}

model DailyEntry {
  id          String      @id @default(uuid())
  challengeId String
  dayNumber   Int
  completed   Boolean     @default(false)
  completedAt DateTime?
}

model Streak {
  id      String      @id @default(uuid())
  userId  String
  challengeId String
  length  Int
  lastUpdated DateTime
}
